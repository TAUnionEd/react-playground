## React Playground

一个 react 构建实验场

### 构建目标

  - 基于 React 16.8.x 的单页应用
  - 使用 Webpack 4 + Babel 7
  - 集成 react-router 与 react-redux
  - 使用 less
----
  - 添加 decorator 语法支持
  - 挂载 ant-d 3.15.x
  - 集成 vuera, 并将 vue 2.6.x 的独立页面挂载至 react-router
  - 升级到 React 17.x


### Run

    $ npm i
    $ npm start
